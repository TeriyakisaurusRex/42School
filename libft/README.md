## ft_isalnum
***is alphanumerical***
### Prototype 
`int ft_isalnum(int c)`
### Return value
returns 1 if the char passed is a digit ('0'-'9') or alphabetical character ('A'-'z'), otherwise returns 0
### Dependancies
- ft_isdigit
- ft_isalpha
### Example:
```c
if (ft_isalnum(c))
  write(1, &c, 1);
```

## ft_isascii
***is ascii***
### Prototype
`int ft_isascii(int c)`
### Return value
returns 1 if the char passed is a ascii value which is (0-127), otherwise returns 0
### Dependancies
- None
### Example: 
```c
if (ft_isascii(c))
  write(1, &c, 1);
```

## ft_isdigit
***is digit***
### Prototype
`int ft_isdigit(int c)`
### Return Value
returns 1 if the char passed is a digit ('0'-'9'), otherwise returns 0
### Dependancies
- None
### Example:
```c
if (ft_isdigit(c))
  int j = c - '0';
```

## ft_isprint
***is printable char***
### Prototype
`int ft_isprint(int c)`
### Return Value
returns 1 if the char passed is printable (ascii 32 - 126), otherwise returns 0
### Dependancies
- None
### Example:
```c
if (ft_isprintable(c))
   write(1, &c, 1);
```

## ft_isspace
***is space char***
### Prototype
`int ft_isspace(int c)`
### Return Value
returns 1 if the char passed is space character (tab, newline, vertical tab, carrage return, return, form feed or space), otherwise returns 0
### Dependancies
- None
### Example:
```c
while (ft_isspace(string[i]))
   i++;
```

## ft_tolower
***to lowercase***
### Prototype
`int ft_tolower(int c)`
### Return Value
If the char passed is uppercase ('A'-'Z') returns the corrisponding lowercase ('a'-'z'), if its not uppercase returns c unchanged
### Dependancies
- ft_isupper
### Example:
```c
char c = 'A';
printf("%c", tolower(c));
```
## ft_toupper
***to uppercase***
### Prototype
`int ft_toupper(int c)`
### Return Value
If the char passed is lowercase ('a'-'z') returns the corrisponding uppercase ('A'-'Z'), if its not lowercase returns c unchanged
### Dependancies
- ft_islower
### Example:
```c
char c = 'a';
printf("%c", tolower(c));
```

## ft_atoi
***array to integer***
### Prototype
`int ft_atoi(const char *str)`
### Return Value
Returns and integer representation of the string passed removing leading spaces, returns 0 on invalid string
### Dependancies
- ft_isspace
### Example:
```c
char s[] = " 123999";
int x = atoi(s);
x /= 2;
printf("%d", x);
```

## ft_itoa
***integer to array***
### Prototype
`char	*ft_itoa(int n)`
### Return Value
Returns a malloced string representation of the integer passed
### Dependancies
- ft_calloc
### Example:
```c
int x = 42;
char *s;
s = ft_itoa(x);
printf("%s", s);
free(s);
```


## ft_memchr
***scan memory for char***
### Prototype
`void	*ft_memchr(const void *s, int c, size_t n)`
### Return Value
returns a pointer to the first occurance of *c* in the array, otherwise returns 0
### Dependancies
- None
### Example:
```c
char s[] = "string";
char c = 'r'
printf("%s", ft_memchr(s, r, 5));
```

## ft_memcmp
***memory compare***
### Prototype
`int	ft_memcmp(const void *s1, const void *s2, size_t n)`
### Return Value
Iterates over the string looking for a differnce in char value between the two arrays, the function will return an int that is the difference in ascii value i.e. 'a' - 'b' = -1
### Dependancies
- None
### Example:
```c
char s1[] = "string";
char s2[] = "strimg";
printf("%d", ft_memcmp(s1, s2, 6));
```

## ft_memcpy
***memory copy***
### Prototype
`void	*ft_memcpy(void *dst, const void *src, size_t n)`
### Return Value
copies *src* into *dst* *n* bytes then returns *dst*
### Dependancies
- None
### Example:
```c
char s1[] = "string";
char s2[] = "strimg";
printf("%d", ft_memcpy(s1, s2, 6));
```

## ft_memmove
***memory move***
### Prototype
`void	*ft_memmove(void *dst, const void *src, size_t len)`
### Return Value
copies *src* into *dst n* bytes **even if they are overlapping in memory** then returns *dst* 
### Dependancies
- None
### Example:
```c
char s1[] = "string";
char *s2 = s1[2];
printf("%d", ft_memmove(s1, s2, 6));
```

## ft_memset
***memory set***
### Prototype
`void	*ft_memset(void *b, int c, size_t n)`
### Return Value
sets *n* bytes of *b* string to the character *c* then returns a pointer to the string
### Dependancies
- None
### Example:
```c
char s1[] = "string";
char c = '!';
printf("%d", ft_memset(s1, c, 3));
```

## ft_putchar_fd
***write char to file descriptor***
### Prototype
`void	ft_putchar_fd(char c, int fd)`
### Return Value
writes the char *c* to the file descriptor (1 for standard output to the console)
### Dependancies
- <unistd.h>
### Example:
```c
char c = '!';
ft_putchar_fd(c, 1);
```

## ft_putnbr_fd
***write number to file descriptor***
### Prototype
`void	ft_putnbr_fd(int n, int fd)`

### Dependancies

- ft_atoi

```c
ft_putnbr_fd(x, 1);
```
